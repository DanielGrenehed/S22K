# S22k/S20K case
I have this one 3d-printed in tpu on a bambulab p2s, added "s20k" text to the top within the slicer. The dimensions are not perfect, but close enough to not bother me too much. I use 3 M4 bolts and nuts on each half to hold the case together and a magsafe plates sticked to the inside of the bottom pieces. The parts are made to be mirrored so that one design can be used to create both sides, I mirrored it in the slicer before priting it.


# Wording?
The bottom would be the same if using 11 or 10 keys on each half, therefore it is called s22k. The top is however currently designed for only 10 keys hence s20k.
